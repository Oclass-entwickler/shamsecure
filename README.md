## الاستخدام المحلي

### الطريقة السريعة (Static)
1. افتح `index.html` في متصفح الويب
2. جميع الصفحات جاهزة للاستخدام

### الطريقة الكاملة (مع API)
1. **ثبّت المتطلبات:**
   ```bash
   npm install
   ```

2. **أنشئ ملف `.env`:**
   ```bash
   cp .env.example .env
   # ثم عدّل القيم في .env
   ```

3. **شغّل الخادم:**
   ```bash
   npm run dev  # للتطوير
   # أو
   npm start    # للإنتاج
   ```

4. **افتح المتصفح:**
   - الموقع: `http://localhost:3000`
   - لوحة التحكم: `http://localhost:3000/admin`
